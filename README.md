


# Key Manager

Manage the encryption and safe decryption of keys.

## Table of Contents

1. [Installation](#installation)
2. [Usage](#usage)
3. [Contributing](#contributing)
4. [License](#license)

## Installation

```bash
$ git clone <repo>
$ chmod +x install.sh
$ ./install.sh
$ vault -h
```

## Usage

```bash
# Create:
$ vault -n [/path/to/key/to/encrypt]

# Retrieve
$ vault -r [/path/to/key/to/encrypt]
```
